Key State Rules

Loading

  • Spinner inside submit button
  • Button text changes to "Processing..."
  • All inputs and reset button disabled
  • Progress hint text below form

Validation Error

  • Red border + red label on offending field
  • Error message below field
  • Summary banner with error count
  • Client-side + server-side validation

Server Error

  • Error card with icon + message
  • "Try Again" button to retry
  • No redirect, stays on same page
  • Handles timeout + network failure

Success

  • Result card slides in below form
  • Green bar (approved) or red bar (rejected)
  • Form stays filled (user can adjust + retry)
  • Reason codes shown as chips
