Breakpoint Rules

Mobile (< 640px)

  • Form fields: single column
  • Metrics grid: 2 columns
  • Container: full width - 16px
  • Header: no glow orb (perf)
  • Score donut: 100px (vs 130px)
  • Buttons: full width stack

Tablet (640px - 1023px)

  • Form fields: 2 columns
  • Metrics: 4 columns
  • Container: max 700px
  • Buttons: side by side
  • Same as desktop layout

Desktop (≥ 1024px)

  • Container: max 700px centered
  • Header glow orb visible
  • Score donut: 130px
  • Timeline connector dots visible
  • Hover states on all interactive
